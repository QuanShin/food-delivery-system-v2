# Food Delivery System

A microservices-based food ordering and delivery system built with:

- Spring Boot microservices
- React web frontend
- Flutter mobile application
- PostgreSQL
- Redis
- Kafka
- Docker
- AWS ECR
- AWS EKS
- Istio
- Terraform
- Prometheus / Grafana
- Splunk

## Planned Microservices
